這個網站是什麼？和柯文哲的官方網站有什麼關係？

這是響應柯文哲陣營推出的「柯文哲野生官網」活動，網站上的資料是使用柯文哲官方提供的往開放資料 API 來取得相關資訊，透過這種方式，每個人都可以用不同的方式來呈現出這些資料。

也因為這樣，這樣的野生官網有一部份「官網」的性質，因為資料是官方提供，但另一方面，卻又不完全是由官方製作。

所以到底什麼是開放資料 API？

所謂開放資料 API 指的是一種可以讓寫程式的人可以用標準的操作方式，取得使用某種通用的規格來表達的資料，因為整個過程都可以經由寫簡單的程式來處理，所以的大家可以自由發揮拿到的資料。

目前政府也有一些資料使用這種方式釋出，筆者也曾利用政府的開放資料 API 製作過失物招領的 Android APP，不過這是第一次看到有候選人個人的開放資料 API，所以就跟風做了一個網站。

你們都是綠的對不對？否則為什麼只幫柯文哲做網站？

冤枉啊！這是因為柯文哲提供了官方開放 API，所以會寫程式和作網站的人可以很輕易地取得柯文哲的相關資料和把資料呈現出來。但連勝文陣營並沒有類似的東西來讓寫程式的人取得資料，甚至連行之多年的 RSS 交換格式檔案也沒有，自然較少人幫他做網站。

如果連勝文也推出了官方 API 或甚至是只要提供 RSS，我立馬也幫他套版一個！
